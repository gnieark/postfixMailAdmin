#postfixMailAdmin
An alternative to postfixadmin.

L'idée est d'en faire un fork beaucoup plus maléable.

Les quelques requetes SQL (lister les users, lister les domaines, les modifier etc....) que doit faire ce GUI seront listées dans le fichier de config et pourront être modifiées à volonté en fonction de votre infrasttructure e-mail.
